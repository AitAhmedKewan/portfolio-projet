Projet LDAP - Gestion des utilisateurs avec TurnKey Linux LDAP

Installation et gestion d’un serveur LDAP pour l’authentification des utilisateurs – BTS SIO SISR

📍 Présentation du projet

Ce projet consiste à mettre en place un serveur LDAP avec TurnKey Linux LDAP Appliance et à intégrer des machines Ubuntu pour l'authentification des utilisateurs via LDAP. Nous avons exploré l’utilisation des commandes principales : ldapadd, ldapmodify et ldapdelete, tout en configurant un domaine LDAP et en gérant les permissions pour permettre l’ajout d’utilisateurs avec des privilèges sudoers.

🧪 Activités réalisées

  • Installation du serveur LDAP via TurnKey Linux LDAP Appliance, avec un domaine défini (dc=XYZ, dc=dom).
  • Création d’utilisateurs avec les commandes ldapadd et ldapmodify pour ajouter des utilisateurs et mettre à jour leurs informations.
  • Configuration du client LDAP sur une VM Ubuntu pour permettre l’authentification des utilisateurs via LDAP.
  • Gestion des permissions pour que les membres d’une unité organisationnelle (OU) soient sudoers et puissent exécuter des commandes avec des privilèges d’administration.
  • Tests d’authentification des utilisateurs LDAP via le terminal Ubuntu.
  • Exploration des schémas LDAP et ajout d’attributs personnalisés dans le serveur LDAP.
  • Analyse des connexions LDAP à l’aide de commandes comme ldapsearch pour vérifier la structure du répertoire et les utilisateurs existants.

🔐 Sécurisation et bonnes pratiques

  • Utilisation des mécanismes de sécurité LDAP pour protéger les informations sensibles des utilisateurs lors de l’authentification.
  • Configuration des permissions sudoers via LDAP pour garantir que seuls les utilisateurs autorisés puissent exécuter des commandes administratives.
  • Utilisation de TLS pour sécuriser les échanges LDAP.
  • Vérification des utilisateurs et groupes avec les commandes getent et ldapsearch pour assurer une gestion centralisée et efficace des utilisateurs.

📊 Résultats obtenus

✅ Serveur LDAP fonctionnel avec ajout et modification d’utilisateurs. 🖥️ Intégration réussie avec les postes Ubuntu pour l'authentification des utilisateurs via LDAP. 🔒 Accès restreint aux utilisateurs sudoers dans le domaine LDAP. 📞 Vérification réussie des informations avec les commandes ldapsearch et getent. ⚙️ Mise en place d’attributs personnalisés dans le schéma LDAP.
